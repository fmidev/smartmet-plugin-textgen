# smartmet-plugin-textgen

Part of [SmartMet Server](https://github.com/fmidev/smartmet-server). See the [SmartMet Server documentation](https://github.com/fmidev/smartmet-server) for a full overview of the ecosystem.

## Overview

The textgen plugin provides an HTTP interface for generating natural-language weather forecast text from SmartMet Server data. Given a location and time range, it produces human-readable weather summaries.

## Dependencies

- [smartmet-library-textgen](https://github.com/fmidev/smartmet-library-textgen) — text generation algorithms
- [smartmet-engine-querydata](https://github.com/fmidev/smartmet-engine-querydata) — forecast data access
